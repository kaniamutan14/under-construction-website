## 🚀 How to Run this Locally

This project is built using vanilla HTML, Tailwind CSS (via CDN), and Vanilla JavaScript, bundled cleanly with **[Vite](https://vitejs.dev/)**.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed before proceeding.

### Installation

1. Clone or download this repository to your local machine:
   ```bash
   git clone <repository-url-here>
   cd <repository-directory>
   ```

2. Install the lightweight development dependencies:
   ```bash
   npm install
   ```

3. Start the Vite development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local link provided in your terminal (usually `http://localhost:5173`).

### Building for Production
To bundle the project for deployment (e.g., GitHub Pages, Cloudflare Pages, Netlify, or Vercel), run:
```bash
npm run build
```
This will compile the optimized files into a `dist/` folder, which is ready to be uploaded to your hosting provider.
